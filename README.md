```

```

### 抓取的节点已分享到github

[节点分享链接](https://github.com/sanzhang007/node_free)


### urls 
存放要抓取link 可解析出网页中节点，不过大部分节点来自sub类网站
```yaml
index: []
urls:
  ########################自己付费节点(备用)###############################
  #
  ########################################################################

  ###########################优质节点#####################################
  - link
  ########################################################################
```
